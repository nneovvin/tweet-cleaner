# Tweet Cleaner (트윗 청소기) ≽(•⩊ •マ≼  

Delete your tweets easily with a Chrome extension.  

---

안녕하세요 **트윗 청소기(aka 트청)** 를 만들었습니다.  
Hi guys, I made **Tweet Cleaner (Tweet Deleter)**.

> 리트윗, 마음 삭제는 지원하지 않습니다.
> Undo retweet and unlike are not supported.
> 
> 삭제된 트윗은 복구되지 않습니다.  
> Deleted tweets **cannot be recovered**.  
>  
> 계정 정지에 대해서는 책임지지 않습니다.  
> I am **not responsible for any account suspension**.

---

# ✨ 특징

- **서버 없음**  
  사용자 데이터를 수집하지 않습니다. 모든 동작이 **크롬 브라우저 내부에서만** 이루어집니다.

- **백그라운드 실행**  
  탭만 켜 두면 다른 작업을 하면서 삭제가 진행됩니다.

- **중단 후 재시작**  
  중단되어도 자동으로 **중단된 부분부터 다시 시작합니다.**

---

# ✨ Features

- **No server**  
  No user data is collected. Everything runs entirely within your **Chrome browser**.

- **Background execution**  
  Leave the tab open and deletion continues while you do other things.

- **Resume after pause**  
  If stopped, it automatically resumes from where it left off.

---

# ⚠️ 주의사항

X 규정상 빠른 속도로 많은 트윗을 삭제하면  
**계정 제한 또는 정지**가 발생할 수 있습니다.

- 하루 **5,000개 미만 삭제 권장**
- 대량 삭제가 필요하면 **아카이브 기반 트청 사용**
- 소량을 **주기적으로 삭제**하거나 백그라운드 실행 추천

---

# ⚠️ Warning

Deleting tweets too quickly or in large volumes may result in  
**account restrictions or suspension** under X's policies.

- Recommended limit: **fewer than 5,000 deletions per day**
- For large-scale deletions, consider **archive-based tools**
- Best for users who want to delete tweets periodically in small batches  
  or simply leave it running in the background

---

# 📦 설치 방법

1. `.zip` 파일 다운로드 후 압축 해제
2. 크롬 주소창에 `chrome://extensions` 입력
3. **압축해제된 확장 프로그램 로드** 클릭
4. 압축 해제한 폴더 선택

---

# 📦 Installation

1. Download the `.zip` file and extract it.
2. Open `chrome://extensions` in Chrome.
3. Click **Load unpacked**.
4. Select the extracted folder.

---

# 📍 사용 방법

1. 트위터(X) 웹 로그인
2. **프로필 → 답글** 화면 이동
3. 크롬 확장 프로그램 아이콘 클릭
4. **로드하기** 클릭 (자동 스크롤 시작)
5. 로드 완료 후 필터 설정 → **삭제하기**

이제 **탭만 열어 두면 자동 삭제**됩니다.

> **문제가 생기면**  
> 크롬 새로고침 또는 재시작 → **Reset** 클릭 → 3번부터 다시 시작

---

# 📍 How to Use

1. Log in to X (Twitter) on the web.
2. Go to **Profile → Replies**.
3. Click the **Tweet Cleaner** extension icon.
4. Click **Load** (auto-scroll starts immediately).
5. When loading finishes, set filters and click **Delete**.

You can now **leave the tab open while deletion runs in the background**.

> **If something goes wrong**  
> Refresh the page or restart Chrome, click **Reset**, then start again from step 3.

Deleted tweets cannot be recovered. I'm not responsible for any account suspension.**


---


## 특징

- **서버 없음** 사용자 데이터를 수집하지 않습니다. 모든 동작이 크롬 브라우저 내에서만 이루어집니다.
- **백그라운드 실행** 탭만 켜 두면 다른 작업을 하면서 삭제가 진행됩니다.
- **중단 후 재시작** 중단되어도 자동으로 중단된 부분부터 재시작합니다.
- RT 및 마음 취소는 지원하지 않습니다.

### Features
 
- **No server** No user data is collected. Everything runs entirely within your Chrome browser.
- **Background execution** Leave the tab open and deletion continues while you do other things.
- **Resume after pause** If stopped, it automatically resumes from where it left off.
- Unretweet and unlike are not supported.
 

---


## 주의사항

X 규정상 빠른 속도로, 많은 양을 삭제하면 계정 제한이나 정지를 당할 수 있습니다.

- 하루 5천 개 미만 권장
- 한꺼번에 대량 삭제가 필요하다면 아카이브 트청을 이용하세요
- 적은 양을 주기적으로 삭제하거나, 틀어 두고 다른 일 하실 분께 추천합니다

### Warning
 
Deleting tweets too quickly or in large volumes may result in account restrictions or suspension under X's policies.

- Recommended limit: fewer than 5,000 deletions per day
- For large-scale deletions, consider using archive-based tools or services
- Best for users who want to delete tweets periodically in small batches or simply leave it running in the background

## 설치 방법

PC 크롬 기준

1. `.zip` 파일 다운로드 후 압축 해제
2. 크롬 주소창에 `chrome://extensions` 입력
3. **압축해제된 확장 프로그램 로드** 클릭 → 압축 해제한 폴더 선택

### Installation
 
Chrome (PC)

1. Download the .zip file and extract it.
2. Open chrome://extensions in Chrome.
3. Click Load unpacked → select the extracted folder.

## 사용 방법

1. 트위터(X) 웹 로그인 후 **프로필 → 답글 화면**으로 이동
2. 크롬 확장 프로그램 아이콘 클릭해서 트윗 청소기 확장 프로그램 열기
3. **로드하기** 버튼 클릭 (클릭과 동시에 자동 스크롤됩니다)
4. 로드 완료가 뜨면 필터 세팅 후 **삭제하기** 클릭
5. 탭만 열어 두고 다른 작업 하셔도 됩니다

> **잘 안 될 때** 크롬 새로고침 or 크롬 재시작 후 우측 상단 **reset** 클릭, 3번부터 다시 시작

### How to Use

1. Log in to Twitter (X) on the web and go to Profile → Replies.
2. Click the Tweet Cleaner extension icon to open it.
3. Click the blue button (auto-scroll will start immediately).
4. When the load complete message appears, set your filters and click the black button.
5. You can leave the tab open while the deletion runs in the background.
 
> **If something goes wrong** Refresh the page or restart Chrome, click **reset** in the top right corner, then start again from step 3.
